# Testing functions

Use this page for the two main functions: `test_shift(...)` and `test_adverse_shift(...)`.
Start here if you are new to the package.

## What you get back

- `test_shift(...)` returns `ShiftResult` with `.statistic`, `.pvalue`, and `.statistic_name`
- `test_adverse_shift(...)` returns `AdverseShiftResult` with `.statistic`, `.pvalue`, and `.direction`

If you need the null distribution or Bayesian output, see the advanced page.
